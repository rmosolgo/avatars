# Avatar Builder

- To work on it locally, run these commands:

  ```bash
  # for initial setup:
  npm install
  npm install -g gulp
  # to start the development server:
  gulp
  ```
- Live on [firebase hosting](http://pcoavatars.firebaseapp.com)
- Based on the [batmanjs-starter](http://github.com/rmosolgo/batmanjs-starter)
